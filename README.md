# kineticEQ

**EN**  
kineticEQ is a Python library for BGK-type kinetic-equation solvers with a high-level `Config` / `Engine` API and C++/CUDA acceleration.

**JP**  
kineticEQ は，高水準の `Config` / `Engine` API と C++/CUDA 高速化を備えた BGK 型運動論方程式ソルバです。
