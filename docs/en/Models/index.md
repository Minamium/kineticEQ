# Models

This section describes both the mathematical models and the numerical realizations implemented in the current code base. BGK1D is the primary production model and supports explicit, implicit, and holo steppers.

- [BGK1D1V](BGK1D1V/index.md): the main operational model
- [BGK2D2V](BGK2D2V/index.md): skeletal implementation only; the Engine path is incomplete
